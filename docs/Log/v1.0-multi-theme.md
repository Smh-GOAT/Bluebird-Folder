# v1.0 — Multi-Theme System 开发日志

**日期：** 2026-03-28  
**阶段：** Bluebird Folder 全局多主题 UI 重构

---

## 开发内容

### 一、目标

为 Bluebird Folder 实现完整的多主题 + 深浅色模式切换系统，消除所有硬编码的 `zinc-*` / `bg-white` 颜色类，通过 CSS 自定义属性统一管理视觉风格。

### 二、实现方式

#### 2.1 Theme Infrastructure

**`src/app/globals.css`**

- 将原来单一 `:root` 块替换为 6 套 `[data-theme][data-mode]` 选择器，覆盖以下组合：
  - `apple light` / `apple dark`：Apple 系风格，灰白底 + 蓝色主色，dark 下深灰 + 高对比蓝
  - `forsion light` / `forsion dark`：天蓝色调 light，深空黑 + 氖蓝 dark
  - `notion light` / `notion dark`：暖米黄纸质感 light，深灰简约 dark

- 每套主题定义的 CSS 变量：
  `--bg`, `--bg-alt`, `--surface`, `--surface-sub`, `--surface-hover`,  
  `--border`, `--border-sub`, `--border-hover`,  
  `--text`, `--text-sec`, `--text-muted`, `--text-subtle`,  
  `--primary`, `--primary-dark`, `--primary-glow`, `--primary-tint`,  
  `--gradient-from`, `--gradient-to`,  
  `--overlay`, `--sidebar-bg`, `--header-bg`, `--panel-shadow`,  
  `--input-bg`, `--chat-ai-bg`, `--chat-ai-border`, `--chat-user-bg`,  
  `--scrollbar`, `--badge-b-bg/fg`, `--badge-pink-bg/fg`, `--badge-green-bg/fg`

- 所有 `.ui-panel`, `.ui-btn-primary`, `.ui-btn-secondary`, `.ui-input`, `.ui-card`, `.ui-bubble-ai` 等组件类全部改为使用 CSS 变量而非硬编码 Tailwind 颜色。

- 新增 `.ui-textarea`、`.ui-nav-item`、`.ui-badge-*`、`.ui-section-label` 等实用类。

**`tailwind.config.ts`**

- 扩展 Tailwind 颜色：`bb-bg`, `bb-surface`, `bb-surface-sub`, `bb-text`, `bb-text-sec`, `bb-text-muted`, `bb-text-subtle`, `bb-primary`, `bb-primary-dark`, `bb-primary-tint`, `bb-border`, `bb-border-sub`, `bb-overlay` 等，值为对应 CSS 变量。
- 新增 `border-radius` tokens (`t-xs` ~ `t-xl`, `t-pill`)、`boxShadow` tokens、`transitionTimingFunction` tokens（Apple / spring / smooth 缓动曲线）。
- 新增动画 keyframes：`fadeUp`, `chipIn`, `pulseDot`。

**`src/lib/theme-provider.tsx`**

- React Context，存储 `theme: "apple" | "forsion" | "notion"` 和 `mode: "light" | "dark"`。
- 挂载时从 `localStorage` 恢复上次设置，并同步 `document.documentElement.dataset.theme/mode`。
- 提供 `setTheme()`, `setMode()`, `toggleMode()` 方法。

**`src/app/layout.tsx`**

- 引入 `<ThemeProvider>` 包裹 `{children}`。
- `<html>` 标签预设 `data-theme="apple" data-mode="light"` 避免 FOUC。

**`src/components/layout/theme-selector.tsx`**

## 技术要点

### CSS 变量 + Tailwind 共用策略
- 直接在 JSX `style` prop 中引用 CSS 变量（如 `style={{ color: "var(--text)" }}`），不依赖 Tailwind 处理 CSS 变量的限制
- Tailwind 自定义颜色（`bb-text` 等）用于需要在类名中组合的场景（模板文件批量替换）

### 主题切换无 FOUC
- `<html>` 有静态默认属性 `data-theme="apple" data-mode="light"`
- `ThemeProvider` 在客户端挂载后从 `localStorage` 还原，用 `applyToHtml()` 直接操作 DOM，不依赖 React re-render

### 排除项（有意保留硬编码颜色）
- `text-rose-600`, `bg-rose-100`：错误状态（语义色）
- `bg-emerald-500`, `text-emerald-700`：成功状态（语义色）
- `text-amber-700`, `bg-amber-50`：警告状态（语义色）
- Template 的内容 accent：cyan/teal/pink/purple/orange 等模板专属色
